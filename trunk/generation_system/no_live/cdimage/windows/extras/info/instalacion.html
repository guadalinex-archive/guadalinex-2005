Instalación

Para iniciar la instalación de Guadalinex en tu disco duro debes hacer doble click en el icono del escritorio Instalación de Guadalinex.

Ten en cuenta que para instalar Guadalinex en tu disco duro sin eliminar el anterior sistema operativo y sin borrar los datos anteriores debes tener una partición libre o espacio suficiente. Las particiones en los discos duros son divisiones de los mismos, que sirven para mantener separados distintos sistemas operativos y/o datos. Un disco duro debe tener, como mínimo, una partición. Si cuando instalaste el sistema dejaste parte del disco sin particionar, esto es, sin asignar a ninguna partición, ahora podrás elegir este espacio para ubicar Guadalinex.

Si no dejaste espacio libre, ¡No te preocupes! Guadalinex es capaz de cambiar el tamaño de la partición que tienes para hacer hueco al final del disco. Si este es el caso deberás arrancar tu otro sistema operativo y “desfragmentar” el disco para que la operación sea posible.

Guadalinex te ayudará durante el proceso de instalación, aunque si prefieres manejar tú mismo el disco puedes usar el particionado integrado.

Una vez que termines el particionado, Guadalinex continuará con la instalación. Los siguientes pasos serán para configurar la red. Aunque no debes preocuparte demasiado por eso ahora, tendrás la oportunidad de cambiarlo todo una vez que reinicies Guadalinex.

El instalador te pedirá también un nombre de usuario y una clave. No las olvides, pues te harán falta para entrar en tu ordenador y para realizar todas las tareas de mantenimiento del equipo. En versiones anteriores era necesario un usuario especial, conocido como root. Ahora, el primer usuario, el que se crea durante la instalación, es considerado el administrador de la máquina, y por tanto tiene privilegios especiales. Es muy importante que conserves la clave en lugar seguro si tu memoria no es muy buena. Guadalinex volverá a pedir esta clave cada vez que haya que realizar una tarea crítica.

Una vez que el instalador tiene todos los datos necesarios, comienza a copiar archivos (puede tardar bastante, según el modelo de ordenador). ¡ Es necesario tener paciencia en este punto !

Cuando finaliza la copia de archivos el sistema se reinicia solo (es el momento de extraer el CDROM). Aparecerá el menú de inicio donde puedes elegir entre los sistemas operativos disponibles en tu disco duro.
