ESTUDIO DE USABILIDAD PARA GUADALINEX 2005
==========================================

Fecha: 26 de Agosto de 2005


Objetivos: 
---------

 - Identificar problemas de usabilidad que se encuentra un usuario de Windows al utilizar un escritorio GNOME.
 - Mejorar en lo posible la usabilidad de Guadalinex 2005.

Estrategia:
----------

 - Estudios de "uso guiado" en el mayor número de participantes como sea posible.

Cómo se hacen las pruebas:
-------------------------

 - El sistema de prueba es un escritorio GNOME v.2.10 (ver apartado "Plataforma
   de pruebas")
 - Los participantes deben realizar una serie de pruebas, que tratan de incluir
   algunas de las tareas más habituales que se llevan a cabo mediante un ordenador.
 - El perfil de los participantes debe ser lo más variado como sea posible, preferiblemente no técnicos y sin o con muy poca
experiencia con Linux.
  
Plataforma de pruebas:
---------------------

- La plataforma de pruebas actual es un sistema Ubuntu Hoary, actualizado a la
versión actual de Breezy. 
- Se ha instalado manualmente el paquete "language-pack-gnome-es-base" (porque el
escritorio aparecía en inglés).
- Se ha incluido en el escritorio el icono "Comenzar Aquí.html", con instrucciones
básicas de cómo comenzar a utilizar el sistema.

Duración prevista
-----------------

30-60 minutos, según perfil.

Referencias
-----------

- http://developer.gnome.org/projects/gup/usertesting.html
- http://primates.ximian.com/~pete/GUADEC/wms-guadec-2005.sxi
- http://developer.gnome.org/projects/gup/ut1_script.html

***************
* Las pruebas *
***************

Cómo deben llevarse a cabo
--------------------------

Las pruebas deben ser tutorizadas por una persona con conocimientos medios/altos
de un sistema Linux.

Esta persona debe encargarse de preparar la plataforma de pruebas así como de
hacer el seguimiento de las pruebas durante su realización. 

Deberá ser esta persona la que se encargue de verificar las respuestas dadas por
el usuario y completarlas con sus propios comentarios.

Para el usuario de las pruebas
------------------------------

A continuación, se va a suponer una situación particular, y se te va a proponer la realización de una serie de pruebas. Cada una de ellas deberás tratar de realizarlas en el menor tiempo posible y, una vez completadas, deberás rellenar varias cuestiones relativas a la prueba y pasar a la siguiente. En caso de que no consigas completar alguna de las pruebas, no te preocupes; contesta a las cuestiones relativas a la misma y pasa a la siguiente.

Intro
-----

"Piensa que ayer fue tu cumpleaños y te han regalado este portátil. En las
próximas cuestiones, vamos a trabajar con él para explorar su contenido:
qué programas se incluyen, cómo se realizan algunas tareas habituales..."
