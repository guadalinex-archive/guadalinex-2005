Function Details

check_password(passwd1, passwd2)

Check the correctness of a proposed password, writen twice.

Return codes:
  • 0 valid.
  • 1 wrong length.
  • 2 strings do not match.

check_username(name)

Check the correctness of a proposed user name.

Return codes:
  • 0 valid.
  • 1 wrong length.
  • 2 contains white spaces.
  • 3 is already taken or prohibited.

invalid_names()

Find out which user names should not be used.

Returns a set of prohibited user names.
