¿ Qué es Guadalinex ?

Guadalinex es un sistema operativo y un conjunto de aplicaciones GNU/Linux, basados en Debian. Es un proyecto de la la Junta de Andalucia que pretende promover el uso del Software Libre para facilitar el acceso a la Sociedad del Conocimiento.

¿ Cómo usar este CD ?

Con este CD puedes evaluar Guadalinex e instalarlo posteriormente en el disco duro de tu ordenador, si así lo deseas. Incluye más de 2 Gigabytes (comprimidos) de software adecuadamente preconfigurado, listo para ser usado: aplicaciones de ofimática, juegos, programas de diseño... Guadalinex detecta una gran cantidad de hardware y se configura automáticamente.

Sólo tienes que insertar el CD en la unidad y arrancar el ordenador. Si tu ordenador no es capaz de arrancar desde la unidad de CD, por favor lee las instrucciones para crear un disquete de arranque.
